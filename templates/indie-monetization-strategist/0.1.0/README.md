# indie-monetization-strategist

Solo developer monetization and product launch strategist for indie SaaS, open-source projects, and one-person businesses.

- Upstream inspiration: [one-person-businesses-methodology](https://github.com/easychen/opc-methodology), [lemonade-stand](https://github.com/nayafia/lemonade-stand), [howto-make-more-money](https://github.com/easychen/howto-make-more-money), [open-saas](https://github.com/wasp-lang/open-saas)
- Spwnr domain: `Specialized Domains`
- Compatibility: `claude_code`, `copilot`

## Summary

Use this agent when you need to design a monetization strategy for an indie software product, open-source project, or one-person SaaS. Invoke for pricing model selection (freemium, usage-based, seat-based), distribution channel planning, open-source monetization approaches (sponsorships, dual license, hosted SaaS), launch sequence design, and early-revenue validation.

Differentiated from `project-idea-validator` (which evaluates ideas): this agent activates after the idea is chosen and focuses on how to make money from it.

## When to Invoke

- Choosing a monetization model for a new product or open-source project
- Designing a pricing page (tiers, amounts, annual vs monthly)
- Planning a product launch sequence (Product Hunt, HN, Reddit)
- Diagnosing why a product isn't converting (funnel analysis)
- Deciding whether to open-source or keep a tool proprietary

## Key Capabilities

- Monetization model selection: match product type to best-fit model (subscription, usage-based, one-time, open-core, dual license, sponsorware)
- Pricing strategy: tiered pricing design, annual discount, price anchoring, value-based pricing
- Distribution planning: GitHub, Product Hunt, Hacker News, dev communities, partnership channels
- Launch sequence: validation phase, launch day checklist, first 90 days MRR growth plan
- SaaS metrics: MRR, churn rate, LTV, CAC, LTV/CAC ratio, trial-to-paid conversion benchmarks
- Anti-patterns: common indie mistakes and how to avoid them

## Dependencies

None — this is a pure advisory agent. No runtime dependencies required.
